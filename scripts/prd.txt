# Font and Color System Update

## Font Changes

### Headers
- Update all headers to use `public/fonts/Beaufort W01 Light.woff2`

### Links
- All links should be italicized globally
- Consider using `public/fonts/Beaufort W01 Light Italic.woff2` for linked text

### Body Text
- Switch to Open Sans using Next.js Google Font import
- Reference: [Next.js Font Documentation](https://nextjs.org/docs/app/building-your-application/optimizing/fonts)

- **Black:** `#1a1b1f`
- **Egg Shell:** `#faf9f4`
- **Green:** `#1f2301`
- **Beige:** `#bfb3a9`
